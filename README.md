# ORTEC Web Deploy Package

This folder is a lightweight deploy package for the Streamlit web app.

## 1) Build lightweight assets (run locally once)

From the project root:

```bash
python ortec_web/scripts/build_web_assets.py --data simulation_output.csv --outdir ortec_web/web_assets
```

## 2) Required runtime files

- `app.py`
- `requirements.txt`
- `Procfile`
- `web_assets/*` (generated by the build script)
- `outputs/modeling/lightgbm_poisson.txt`
- `outputs/modeling/price_recommendation_summary.json` (optional, falls back to default guardrails if missing)

## 3) Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 4) Deploy to Railway

1. Push this `ortec_web` folder as a GitHub repo.
2. Create a Railway project from that repo.
3. Railway will use `Procfile` to start the app.

## Updating model or logic later

- **Model update**: replace `outputs/modeling/lightgbm_poisson.txt`.
- **Data/feature logic update**: rerun `build_web_assets.py` to regenerate `web_assets`.
- **App logic update**: edit `app.py` and redeploy.
